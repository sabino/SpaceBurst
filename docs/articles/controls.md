# Controls

## Desktop

- `W A S D`: move
- `Arrow Keys`: aim
- `Space`: fire
- `Q / E`: rotate owned weapon styles
- `R`: rewind
- `Esc`: pause
- `F1`: help

## Gamepad

- Left stick: move
- Right stick: aim
- Right trigger: fire
- Left shoulder: rewind
- D-pad left/right: rotate owned weapon styles

## Android

- Left touch pad: move
- Right touch pad: aim and fire
- `R` touch button: rewind

## Tutorial

The first run starts with a tutorial prologue. It teaches movement, aiming, firing, rewind, power-core collection, style switching, lives versus ships, and the upgrade draft flow.
